Image Gallery Web App

A clean and responsive Image Gallery built using HTML, CSS, and JavaScript.
This project showcases a modern UI design with smooth user interaction and organized image display.

---

 Features

-  Responsive grid layout
-  Image preview / enlarge functionality
-  Modern and clean UI design
-  Fast loading and smooth performance
-  Mobile-friendly design

---

 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
